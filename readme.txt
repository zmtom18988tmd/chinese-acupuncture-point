分享一个自己做的开源 穴位模型图。

解压，然后用 blender打开它（blender版本为blender2.92）

使用方法为：
鼠标 点击 一个 穴位 立方体， 然后在 右边的 脚本区域，
选择脚本 show（默认已经选择好了，不用管）
然后点击 show 右边的 三角形 按钮 就会显示 穴位的信息。
